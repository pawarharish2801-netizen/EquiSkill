EquiSkill

Lean Canvas  ·  GenAI Career Assistant  ·  July 2026

Problem
  • GenAI skill training is expensive or gated behind bootcamps ($500–$5 000+)
  • Interview prep, resume writing, and job search are fragmented across many paid tools
  • Professionals in developing regions lack free, high-quality AI mentorship
  • Learners don't know which GenAI skills employers actually want
Existing Alternatives
  • Coursera / Udemy (paid, passive)
  • ChatGPT (generic, no career focus)
  • LinkedIn Learning (subscription)
  • Resume.io (paid, no AI interview)
Solution
6 Integrated Modes
  • 📘 Tutorial Generator — live Tavily web-search + Gemini to write up-to-date GenAI tutorials with code
  • 💬 Ask a Question — expert GenAI engineer persona Q&A
  • 📝 Interview Prep — curated Q&A with web-sourced references
  • 🎤 Mock Interview — resume-aware AI interviewer
  • 📄 Resume Builder — conversational → professional LaTeX PDF
  • 🔍 Job Search — live listings + clean Markdown summary

Tech Stack

Gemini (5-model fallback) · LangChain · FastAPI · Streamlit · MySQL · Docker

Unique Value Proposition

The only free, all-in-one GenAI career toolkit — learn, practise, build a resume, and find a job, in one place.

No subscription. No signup needed.
Open-source. Self-hostable.
Aligned with UN SDG 4 — Quality Education.

Unfair Advantage
  • 5-model auto-rotating fallback — near-zero downtime even on free API tiers
  • Resume → Interview loop — uploads your CV and quizzes you on your own experience
  • Live Tavily search — tutorials are always current, not stale training data
  • Fully open-source — community-extensible
